# 星敏-数字员工 (XM-IM ERP)

## AI 驱动的全渠道客户运营与业务协同平台

以 IM 为统一交互入口，连接 AI 智能体、工作流、CRM、商品、报价、订单与多渠道客户触点

---

## 🎯 企业客户运营的挑战与痛点

- **响应滞后**：7x24 小时服务成本高，首答慢导致流失。
- **渠道分散**：官网、H5、二维码、微信、企微、WhatsApp 等入口各自为战。
- **业务断层**：聊天、客户、商品、报价、订单、售后分散在不同系统。
- **知识难复用**：经验依赖资深员工，新人培训慢，回答质量不稳定。
- **数据不可控**：客户来源、咨询过程、AI 成本、服务质量缺少统一追踪。

---

## 💡 我们的定位与核心价值

### 星敏-数字员工：让企业拥有可配置、可执行、可迭代的 AI 业务员工

1. ⚡ **即时响应**：AI 首答、知识问答、意图识别与自动澄清。
2. 🧠 **业务自动化**：工作流驱动咨询、商品推荐、询价、报价、下单。
3. 🤝 **人机协同**：低置信度、投诉、金额阈值等场景无感转人工。
4. 📊 **数据闭环**：客户资产、会话记录、工作流执行、AI 调用成本可追踪。
5. 🔌 **生态连接**：通过系统对接配置，将外部 API 纳入 AI 工作流。

## 🗺️ 平台能力地图

- **🧠 AI 智能体引擎**：百炼/通义千问接入，流式回复、公共提示词、运行时上下文注入。
- **⚙️ 工作流 DSL 引擎**：意图分类、字段收集、条件分支、API 调用、发卡片、转人工。
- **📦 商品与价格能力**：商品主数据、SKU、价格规则、客户标签/类型差异化报价。
- **💬 自研 IM 通信**：WebSocket 实时消息、历史拉取、文件/卡片消息扩展。
- **👥 CRM 与渠道好友**：客户资料、标签、负责人、渠道好友绑定与客户主档整合。
- **🔌 系统对接配置**：租户级外部 API 档案、鉴权配置、工作流 `integrationCode` 复用。
- **🛡️ 多租户与权限**：tenant_id 全链路隔离、RBAC、数据权限、操作日志。
- **🧰 标准后台能力**：用户、角色、权限、字典、参数、OSS 文件、定时任务。

## 🚀 端到端业务闭环

客户咨询 ➡️ AI 理解需求 ➡️ 工作流执行业务 ➡️ 人工接管 ➡️ 客户资产沉淀

1. 客户通过官网、H5、二维码、插件入口进入，自动识别来源与业务入口。
2. AI 根据知识库、商品主数据、历史上下文进行欢迎、澄清与推荐。
3. 工作流自动执行商品检索、客户信息收集、价格试算、外部 API 调用。
4. 复杂问题触发转人工，人工可查看 AI 摘要、会话上下文与客户资料。
5. 处理结果沉淀为客户标签、商品意向、报价记录、订单线索与服务数据。

## 🧠 AI 智能体：从“会聊天”到“会办事”

- **运行时上下文**：智能体配置、工作流状态、知识库、商品摘录、公共约束共同注入。
- **受控输出**：通过平台级提示词约束 SKU、报价、卡片、文件等关键业务边界。
- **转人工策略**：支持置信度、关键词、意图、金额、敏感投诉等规则触发。
- **AI 生成配置**：运营人员用自然语言描述业务，生成 `systemPrompt + workflow_config` 草稿。
- **配置规范化**：保存前对工作流 JSON 做确定性修复，降低模型输出不可执行风险。

---

## ⚙️ 工作流引擎：企业流程的可执行大脑

- **节点编排**：分类、收集字段、条件判断、变量设置、API 调用、发消息、发卡片、转人工。
- **状态管理**：会话级工作流状态、已收集字段、变量上下文、步骤日志可追踪。
- **业务复用**：商品、报价、订单、客户标签、外部系统调用通过标准节点沉淀。
- **低代码运营**：面向运营配置，不要求每个租户定制开发。
- **可迭代优化**：支持后续接入测试评分、AI 草稿优化、灰度发布与回滚。

## 📦 商品、价格与报价能力

- **商品主数据**：分类、SPU、SKU、图片、扩展属性、资料下载统一管理。
- **商品介绍节点**：AI 可基于在售商品向客户做安全介绍，不暴露成本、供应商等敏感字段。
- **商品意向节点**：将客户关注的 SKU、数量、用途沉淀为“类购物车”，服务后续报价/订单。
- **价格规则**：按客户类型、标签、SKU、生效时间、优先级配置差异化价格。
- **报价试算**：IM 场景可调用统一算价入口，保障报价口径一致。

## 👥 客户资产与渠道好友整合

- **客户主档**：客户资料、标签、负责人、来源渠道、互动历史统一归档。
- **渠道好友**：设备账号、企微/插件同步好友与 CRM 客户可绑定、归入企业客户。
- **单一入口**：原“好友管理”能力并入“客户信息”，降低培训与权限维护成本。
- **客户规则**：支持客户分组、标签、运营规则与后续自动化动作扩展。
- **长期价值**：客户不再只是聊天对象，而是可运营、可追踪、可转化的数字资产。

## 🔌 系统对接与自动化能力

- **对接档案**：按租户维护外部系统 Base URL、鉴权、Header、超时、环境等配置。
- **安全复用**：工作流通过 `integrationCode` 引用档案，避免密钥写入工作流 JSON。
- **API 调用节点**：把 CRM、ERP、招聘、物流、支付等第三方系统接入 AI 流程。
- **定时任务**：后台可配置 Cron，支持 SQL 脚本、内置处理器、立即执行与执行日志。
- **参数/字典管理**：系统配置集中化，支持字符串、数字、布尔、JSON 等类型。
