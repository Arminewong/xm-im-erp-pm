# 星敏-数字员工
通用行业版（V2.0）

副标题：AI驱动的全渠道业务协同平台  
关键词：IM｜AI 智能体｜工作流｜多租户｜微信/企微

---

# 一句话定位
- 基于 IM 的企业协同与客户运营平台：AI 智能体 + 人工坐席协作，覆盖获客、咨询、报价、下单与售后全链路

---

# 核心价值主张
- 7x24 智能应答：AI 首屏 < 2s
- 降本增效：AI 处理标准问题 + 工作流自动化
- 业务闭环：咨询 → 报价/订单/工单，数据全留痕、可质检
- 多渠道统一：链接、二维码、微信/企微插件统一接入

---

# 能力地图（摘要）
- AI 智能体引擎（百炼对接｜知识库｜多模态预留）
- 自研 IM 通信（WebSocket 实时消息、文件上传）
- 知识库管理（QA 对、上下文注入，RAG 预留）
- 工作流引擎（意图判断、API 调用、分支、转人工）
- 插件入口与渠道适配（微信/企微签名、OAuth、会话初始化）
- 多租户与权限（tenant_id 强隔离，JWT）
- 部署与运维（Nginx + Spring Boot + Redis + MySQL）

---

# 典型流程（概览）
1. 入口访问（二维码/链接/插件） → 校验与建联
2. AI 首次响应 + 知识/流程辅助
3. 触发工作流（提问参数 → 调接口 → 生成卡片）
4. 转人工策略命中 → 人工接入
5. 业务闭环（报价/订单/工单）+ 质检与报表

---

# 模块功能清单（1/2）
- 客户端（H5/小程序）：自动登录、历史消息、AI 流式打字、图片/文件发送、弱网重连
- 会话与消息：会话状态机（ai_serving/waiting/human_serving/closed）、消息幂等与 ACK、离线消息拉取
- AI 智能体：模型参数、Prompt、知识库关联、流式输出（ai_typing）、转人工规则
- 知识库：知识库/QA 对 CRUD、上下文注入、RAG 对接预留

---

# 模块功能清单（2/2）
- 工作流：意图判断、API 调用、消息发送、条件分支、转人工（可视化规划）
- 入口管理：入口编码、类型（qrcode/link/embed）、欢迎语、菜单按钮、AI 绑定
- CRM：客户档案、标签、互动历史
- 报价模板：模板配置、公式、PDF 输出（卡片消息二期）
- 订单/工单：单据生命周期（按迭代完善）

---

# 架构与接口（摘要）
- 后端：Spring Boot + MyBatis-Plus + Redis + MySQL
- 客户端：uni-app + Vue3（H5/小程序）
- 接口前缀：/api/admin；IM 业务：/v1/im/**；WebSocket：/api/admin/ws/im?token=JWT
- 插件端：/v1/plugin/**（entry/init 等）

---

# 安全与合规
- 全链路 HTTPS/WSS
- 多租户隔离（SQL 注入 tenant_id）
- 敏感数据加密（手机号等）
- 语音/文件合规（用户授权）

---

# 部署与运维
- 单机：Nginx + Spring Boot（8081）+ 本地 Redis + 远程 MySQL
- 脚本：build → upload → deploy（systemd）
- 可演进：SLB + 多实例、Redis 主从/集群、RDS、MQ

---

# 路线图（MVP → 二期）
- MVP：AI 对话、IM 基础消息/文件、入口管理、管理后台
- 二期：语音/视频/卡片、工作流可视化、RAG 检索、质检报表、消息分表与已读回执

---

# KPI 模板

- 报价转化率：按入口与行业细化
- 人均会话承载能力：≥ 3 倍提升
