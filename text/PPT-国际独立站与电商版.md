# 星敏-数字员工
国际独立站 & 电商行业版（V2.0）

副标题：覆盖获客→咨询→报价→下单→售后全链路的 AI + IM 方案

---

# 行业洞察
- 独立站流量获取成本高，转化率敏感
- 时差与语言差异带来的咨询响应挑战
- 报价/物流/售后协同链条长，流程割裂

---

# 一句话方案
- 在网站/社媒/广告着陆页接入 IM 客服入口，AI 7x24 首响 + 工作流驱动询价/下单，必要时无感转接人工，统一沉淀咨询与商机

---

# 入口与渠道
- 二维码/短链嵌入广告、着陆页、邮件、WhatsApp 网页端（可扩展）
- 微信/企微插件入口：签名 + OAuth + 会话初始化（entryCode 绑定渠道）  
- UTM/scene 参数：来源渠道统计与转化归因（规划/可扩展）

---

# 电商/独立站典型流程
1. 客户点击广告进入落地页 → 打开 IM 入口
2. AI 首问引导 + 询问关键参数（型号/数量/收货地等）
3. 调用价目表/物流 API → 生成报价卡片
4. 客户确认 → 生成订单/工单（或转人工跟进）
5. 售后咨询 → 工单闭环 + 质检

---

# 能力清单（行业定制）
- AI 智能体：多语言友好（Prompt/知识库），跨时区 7x24
- 报价卡片（规划/二期）：规格/数量/总价，支持确认/改量
- 物流/时效咨询：与内部系统/三方 API 对接（工作流“调用 API”节点）
- 线索沉淀：客户档案、标签（高意向/频次/来源）
- 数据看板：入口渠道漏斗与转化（报表中心）

---

# 模块功能（映射）
- 客户端（H5/小程序）：IM 聊天、文件/图片、历史消息
- 会话与消息：状态机 + ACK + 离线拉取
- 工作流：意图判断/收集参数/调用接口/生成卡片/转人工
- 知识库：FAQ、物流政策、退换货规则等
- 入口管理：按渠道/活动创建 entryCode，欢迎语与菜单定制
- 企微员工移动端：销售/客服移动接入会话与消息中心

---

# 架构与接口（摘要）
- 接口前缀：/api/admin；IM：/v1/im/**；WebSocket：/api/admin/ws/im?token=JWT
- 插件入口：/v1/plugin/entry/init（签名 + OAuth + 会话初始化）
- 文件上传：/v1/im/file/upload
- 历史消息：/v1/im/message/history

---

# 安全与合规（跨境关注）
- 全站 HTTPS/WSS，访问控制合规
- 客户数据与聊天内容按需加密存储
- 多租户隔离，按租户统计与报表

---

# 部署建议
- 单机上线：Nginx + Spring Boot + Redis + MySQL（脚本化部署）
- 可演进：SLB + 多实例、RDS、Redis 主从/集群、MQ
- CDN：静态资源与海外加速（可选）

---

# 业务KPI（建议）
- 首响时长：AI < 2s
- 转化指标：咨询→报价→下单漏斗（分渠道/活动）
- 人工介入率与AI解决率
- 客单价与售后工单率

---

# 行业案例（占位）
- 跨境物流：OCR/文档解析辅助询价，报价卡片 + 工单闭环
- 独立站电商：广告落地页转化，AI 收集参数 + 报价/下单工作流

---

# 路线图（行业版）
- MVP：入口/AI应答/IM消息/文件上传/后台配置
- 二期：报价卡片/订单工单联动、工作流可视化、RAG 检索与多语言优化、渠道漏斗看板
