## 一、官网定位

### 1.1 官网目标

- 让首次访问者在 15 秒内看懂“星敏数字员工”是什么
- 让潜在客户在 1 分钟内看见与自身业务相关的应用场景
- 让采购决策者快速理解系统能力、部署方式与安全边界
- 让销售线索更自然地进入“体验 / 咨询 / 留资”流程

### 1.2 核心定位

- 星敏数字员工是一套基于 IM 的企业协同与客户运营平台
- 通过 AI 智能体 + 人工坐席 + 工作流引擎，覆盖获客、咨询、报价、下单与售后全链路
- 适合需要“实时接待、知识问答、线索收集、业务协同、转人工闭环”的企业

### 1.3 官网调性

- 关键词：专业、可信、易懂、业务导向、可落地
- 避免过度技术化，不堆砌模型名词
- 先讲业务价值，再讲能力支撑，最后讲部署与安全

### 3.5 工作流程展示

**区块标题**
- 从收到消息到完成业务，形成标准化闭环

**五步流程**
1. 客户从官网、二维码、链接或插件入口进入会话
2. AI 首次响应，识别客户意图并调用知识库辅助作答
3. 根据业务规则主动追问关键信息，触发工作流
4. 系统生成报价、工单或业务结果，复杂问题转人工接管
5. 全流程留痕，进入质检、统计与持续优化

**展示文案**
- 让每一条咨询都有响应、每一次沟通都有沉淀、每一个商机都有去向

### 3.9 部署与安全区

**区块标题**
- 可快速上线，也能满足企业级安全要求

**文案建议**
- 支持单机快速部署：Nginx + Spring Boot + Redis + MySQL
- 支持后续平滑演进：SLB、多实例、Redis 主从/集群、RDS、MQ
- 多租户隔离：共享库 + tenant_id 强隔离
- 统一鉴权：JWT + 租户上下文
- 全链路安全：HTTPS / WSS、敏感数据加密、统一签名与错误码规范
